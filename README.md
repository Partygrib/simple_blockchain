## Тест
Создание небольшого простого блокчейна для курса по Разработке сетевых приложений.

## Как запустить?  
### Используя Docker:  
Запустите терминал в папке проекта.   
Введите следующие команды: 

>docker build -t simple-chain .    
>docker-compose up
